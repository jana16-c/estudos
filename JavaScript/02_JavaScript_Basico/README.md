# JavaScript Fundamentals

JavaScript is a versatile programming language used primarily for web development. It allows you to create interactive and dynamic web applications. Here are some fundamental concepts:

1. **Variables**: Used to store data values.
   - `let`, `const`, and `var` are used for declaring variables.

2. **Data Types**: JavaScript provides several data types, including:
   - Numbers: `let num = 5;`
   - Strings: `let str = "Hello, World!";`
   - Booleans: `let isTrue = true;`

3. **Functions**: A block of code designed to perform a particular task. Functions can be defined using:
   - Function declarations
   - Function expressions

4. **Control Structures**: Conditional statements (if, else) and loops (for, while) allow for decision-making and repeated execution.

5. **Objects and Arrays**: JavaScript supports complex data structures.
   - Objects: `let obj = { key: 'value' };`
   - Arrays: `let arr = [1, 2, 3];`

## DOM Manipulation

The Document Object Model (DOM) represents the structure of a web page. JavaScript can interact with the DOM to modify the document structure, style, and content. Key methods include:

- `document.getElementById()`: Selects an element by its ID.
- `document.querySelector()`: Selects the first element matching a CSS selector.
- `element.innerHTML`: Changes the content of an element.
- `element.style`: Modifies the styles of an element.

These methods allow developers to dynamically update the user interface based on user interactions or other events.
