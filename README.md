# JavaScript Study Guide

## 1. HTML/CSS Basics
- Understand HTML structure and syntax.
- Learn about HTML elements, attributes, and tags.
- Familiarize yourself with CSS selectors and properties.
- Understand the importance of responsive design and media queries.
- Explore CSS layouts: Flexbox and Grid.
- Understand box model, margins, padding, and borders.
- Practice creating semantic HTML.
- Explore common HTML forms and inputs.

## 2. DOM Manipulation
- Understand what the DOM (Document Object Model) is.
- Learn how to select DOM elements using `document.querySelector` and `document.getElementById`.
- Practice creating, removing, and modifying DOM elements.
- Understand event handling and listeners.
- Learn about event propagation (bubbling and capturing).
- Explore different event types (click, input, change, etc.).
- Understand the importance of the `this` keyword in event handlers.
- Practice manipulating styles and attributes of DOM elements.

## 3. Data Logic with Regex and Arrays
- Understand what regular expressions (regex) are and their use cases.
- Learn basic regex syntax and patterns.
- Practice creating regex for validation (emails, phone numbers, etc.).
- Get familiar with JavaScript arrays: creation, accessing, and manipulation.
- Learn about array methods: `push`, `pop`, `shift`, `unshift`, `map`, `filter`, `reduce`.
- Practice using array methods for data transformation.
- Understand the difference between mutable and immutable operations on arrays.

## 4. Async Operations with Promises
- Understand the importance of asynchronous programming in JavaScript.
- Learn about callback functions and their limitations.
- Understand what promises are and how to use them.
- Practice creating and resolving promises.
- Learn about `Promise.all` and `Promise.race`.
- Understand the use of `async/await` syntax for handling promises.
- Learn how to handle errors in asynchronous operations.

## 5. Dynamic Rendering
- Understand what dynamic rendering is and why it’s important.
- Learn about client-side rendering vs server-side rendering.
- Practice using JavaScript to dynamically update the DOM based on user input.
- Explore template literals and their use for rendering dynamic content.
- Understand the use of frameworks/libraries (like React) for dynamic rendering.
- Practice building a simple interactive application that utilizes dynamic rendering concepts.
