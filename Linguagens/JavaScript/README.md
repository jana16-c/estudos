## Trilha: Processador de dados no navegador

Esta trilha foi organizada para conectar interface web, lógica de limpeza de dados e renderização de resultados, com foco no cenário de leitura/tratamento de dados vindos de PDF.

## Ordem de estudo

1. `01_Fundamentos_Interface`
2. `02_JS_DOM`
3. `03_Logica_Dados`
4. `04_Assincronicidade`
5. `05_Renderizacao_Saida`
6. `06_Projeto_Guiado`

## Meta final

Construir uma ferramenta web que:
- receba arquivo;
- extraia texto;
- limpe e estruture os dados;
- exiba em tabela;
- permita copiar/exportar resultado.
