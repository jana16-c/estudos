# Fase 1: Fundamentos da Interface (HTML e CSS)

## Objetivo
Criar a base visual e estrutural da ferramenta para entrada e exibição de dados.

## Conteúdo

### HTML básico para ferramentas
- Semântica de formulários e entradas:
  - `<input type="file">`
  - `<input type="number">`
  - `<input type="radio">`
  - `<button>`
- Uso de `<div>` com `id` e `class` para organização e futura manipulação via JavaScript.

### CSS estrutural
- Variáveis no `:root` para tema e consistência (ex.: `--bg`, `--text`, `--primary`).
- Flexbox para alinhamento de grupos de controles (`display: flex`, `justify-content`, `align-items`, `gap`).
- Estados interativos essenciais:
  - `:hover`
  - `:disabled`
  - foco visível em campos e botões.

## Checklist
- [ ] Criar formulário básico com upload de arquivo
- [ ] Criar campos numéricos e opções com radio
- [ ] Criar botões de ação (processar, limpar, copiar)
- [ ] Organizar layout com Flexbox
- [ ] Aplicar variáveis CSS no `:root`
- [ ] Definir estilos para `:hover` e `:disabled`
- [ ] Garantir legibilidade em desktop e notebook

## Entregável da fase
Uma página estática funcional (sem lógica de processamento ainda), com todos os campos necessários para o fluxo.
