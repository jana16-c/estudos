# Fase 6: Projeto guiado de consolidação

## Objetivo
Consolidar todas as fases em um projeto único, com organização de código e critérios de qualidade.

## Escopo mínimo do projeto
- Upload de PDF.
- Extração de texto por página.
- Limpeza e estruturação de registros.
- Renderização em tabela.
- Ação de copiar resultado para planilha.

## Organização sugerida
- `index.html`
- `styles.css`
- `main.js`
- `parser.js`
- `ui.js`
- `utils/date.js`
- `utils/text.js`

## Checklist final
- [ ] Separar responsabilidades por arquivo/módulo
- [ ] Manter funções pequenas e nomeadas por intenção
- [ ] Cobrir casos de erro mais comuns
- [ ] Garantir que o fluxo completo funciona do início ao fim
- [ ] Documentar como executar o projeto localmente
- [ ] Registrar limitações e próximos passos

## Critérios de conclusão
- O usuário consegue carregar um arquivo e obter saída consistente.
- O código é legível, modular e fácil de evoluir.
